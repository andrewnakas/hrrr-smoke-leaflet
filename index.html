HRRR Smoke Forecast

Leaflet-based forecast viewer for NOAA HRRR-Smoke. The map stays interactive while the selected NOAA smoke frame animates in the viewer.

How it works: this app parses NOAA’s HRRR-Smoke page for the latest runtime, then builds forecast frame URLs.

Note: NOAA does not expose a clean CORS-friendly public tile API here, so the forecast frame is shown in an embedded viewer alongside the Leaflet map.
